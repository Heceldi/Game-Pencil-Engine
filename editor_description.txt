Welcome to Game Pencil Engine - Version 1.13 Release Candidate!


Game Pencil is a full-featured open source cross-platform 2D game engine and IDE(Integrated Development Environment. It's current version 1.12 specializes in two platforms HTML5 and WiiU exports. It gives a pleasant
experience to making fully functional games with its simple and yet advanced structure.

This new version is the second release since the IDE was made open source with hopes of a bright future ahead in the FLOSS world.

There are giant code refactors in this version of Game Pencil as well as many bug fixes.

We hope you enjoy testing this version and we hope to fix any new problems for a solid 10/29/2017 release of 1.13.
